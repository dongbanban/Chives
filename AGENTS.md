## Agent skills

### Issue tracker

Local markdown in `.scratch/<feature-slug>`. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the canonical five: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

单上下文 — 一个 `CONTEXT.md` + `docs/adr/` 在仓库根目录。See `docs/agents/domain.md`.
